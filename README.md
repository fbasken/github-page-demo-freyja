# Freyja Basken Portfolio

A modern, dark-mode portfolio site built with Vanilla HTML/CSS/JS.

## Features
- **Modern Aesthetic:** Dark theme with neon accents and clean typography.
- **Responsive Design:** Optimized for mobile and desktop.
- **Automated Deployment:** Configured with GitHub Actions for seamless updates to GitHub Pages.

## How to Deploy
1. **Push to GitHub:** Create a new repository on your GitHub account (`fbasken`) and push this directory's contents.
2. **Enable GitHub Pages:**
   - Go to **Settings > Pages**.
   - Under **Build and deployment > Source**, select **GitHub Actions**.
3. **Trigger Workflow:** The deployment will happen automatically on every push to the `main` branch.

## Directory Structure
- `index.html`: Main content and structure.
- `style.css`: Styling and themes.
- `script.js`: Interactive elements.
- `.github/workflows/deploy.yml`: GitHub Actions deployment pipeline.
